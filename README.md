## 📊 EMERGENCY DEPARTMENT DASHBOARD (Power BI)

🔍 **Dataset Overview**  

Rows: 2000  
Columns: 14  
Fields:  
* Patient ID
* Patient Name
* Visit Date
* Race
* Department Referral
* Admission Status
* Wait Time (min)
* Satisfaction (out of 10)
* Admin-Flagged Cases
* Gender
* Age
* Age Band (Calculated Field)
* Day of Week Number (Added)
* Day of Week (Added)

 🛠 **Tools & Technologies**  
 
* Power BI
